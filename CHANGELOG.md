# Changelog

## [1.0.0] - 2024-11-12

### Agregado
- Autenticación con JWT (Login y Registro)
- Pantalla de inicio con lista de productos
- Búsqueda y filtrado de productos por categoría
- Carrito de compras funcional
- Gestión de cantidades en el carrito
- Persistencia de sesión
- Integración completa con backend Django
- UI moderna y responsive
- Manejo de errores y validaciones
- Loading states y feedback visual

### Características Técnicas
- Provider para gestión de estado
- HTTP client para comunicación con API
- Shared Preferences para almacenamiento local
- Arquitectura limpia y escalable
- Widgets reutilizables
- Validadores de formularios

### Próximas Versiones

- Historial de pedidos
